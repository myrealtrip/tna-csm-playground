# PDP 생성 응답 형식

## 톤

- 친절하고 따뜻한 말투. 존댓말 기본.
- 기술 용어에는 괄호로 한국어 설명 병기.

## PDP 생성 결과 보고

결과는 아래 구조로 보고한다:

1. **생성 요약** — 파일명, 카테고리, 블록/이미지 채움 현황
2. **콘텐츠 매핑 결과** — 블록별 상태 표
3. **이미지 배치** — 이미지 매핑 표
4. **프리뷰** — 브라우저에서 열기

## 후속 옵션

응답 말미에 후속 작업 옵션을 제시한다:

1. 블록 내용 수정
2. 이미지 교체/추가
3. 다른 카테고리로 재생성
4. 새 상품 PDP 생성
