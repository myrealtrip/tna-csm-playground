📊 Sendbird 파트너 분석기

파트너 대화 이력을 자동 분석해 확정률·취소율·응대품질을 리포트로 다운로드합니다. AI 없이 브라우저에서 바로 실행됩니다.

STEP 1

북마클릿 설치

아래 버튼을 드래그해서 즐겨찾기 바에 올려놓으세요. 한 번만 하면 됩니다.

⬇️ 이 버튼을 즐겨찾기 바로 드래그

📊 파트너 분석
⚠️ 클릭하면 작동 안 해요. 즐겨찾기 바로 드래그해서 저장하세요.
STEP 2

사용 방법

1
Sendbird 대시보드(dashboard.sendbird.com)에 로그인
2
즐겨찾기에서 📊 파트너 분석 클릭
3
User ID와 분석 기간 입력 (예: P151883)
4
자동으로 sendbird-report-{UserID}.html 다운로드
5
파일을 열면 분석 리포트 + Claude 분석 연결 버튼
