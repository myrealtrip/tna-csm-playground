사용 방법

설치 후 매번 이렇게 사용하면 돼요.

1
Sendbird 대시보드(dashboard.sendbird.com)에 로그인
2
즐겨찾기에서 📨 Sendbird 추출 클릭
3
팝업에 User ID 입력 후 확인
4
자동으로 sendbird-{UserID}.md 다운로드
💡 User ID 찾는 방법

2 우측 상단 파트너 프로필 아이콘 클릭
3 우측 패널에서 User ID 확인 (예: P151883)
