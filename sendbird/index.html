STEP 2

사용 방법

1
Sendbird 대시보드(dashboard.sendbird.com)에 로그인
2
즐겨찾기에서 📊 파트너 분석 클릭
3
User ID와 분석 기간 입력 (예: P151883)
4
자동으로 두 파일 다운로드:
  • sendbird-report-{UserID}.html — 수치 분석 리포트
  • sendbird-{UserID}.md — 전체 대화이력 (Claude Code 분석용)
💡 User ID 찾는 방법
1 Group Channels에서 대화 하나 클릭
2 우측 상단 파트너 프로필 아이콘 클릭
3 우측 패널에서 User ID 확인 (예: P151883)
