simol-diary/entries 폴더를 연결해주세요

사용법
1
아침 (모닝 탭) — 출근 직후 작성. 내용은 자동 저장됩니다.
작성 항목 상세 보기
미래 시각화 — 오늘 향해 가는 방향. "이번 주 발표를 성공적으로 마친 나"처럼 이미 이룬 모습을 상상하며 적어요.
정체성 — "나는 ___하는 사람이다." 되고 싶은 정체성을 매일 선언합니다. 반복하면 뇌가 믿기 시작해요.
내적동기 — 오늘의 해석 프레임. 외부 보상이 아닌, 내가 스스로 의미를 부여하는 이유를 적어요.
감사일기 — 3가지 감사할 일. 크지 않아도 됩니다. 감사 근육을 키우는 훈련이에요.
기상 직후 할 일 — 어젯밤 이브닝에서 적은 내용이 자동으로 표시됩니다.
Brain Dump — 머릿속에 떠오르는 모든 것을 쏟아내세요. 완료하면 체크.
Big 3 — 오늘 가장 중요한 3가지. 이것만 하면 오늘은 성공한 하루입니다.
2
낮 (데이 탭) — 수시로 확인하며 하루를 운영합니다.
작성 항목 상세 보기
Big 3 진행 — 모닝에서 정한 Big 3의 진행 상황을 체크해요. 달성률 프로그레스 바가 표시됩니다.
Time Box — 30분 단위로 실제로 무엇을 했는지 기록합니다. 현재 시간이 하이라이트되어 있어요. 계획이 아니라 실제 한 일을 적는 게 핵심.
Brain Dump — 모닝에서 쏟아낸 항목을 처리하거나, 새로 떠오른 것을 추가하세요.
3
저녁 (이브닝 탭) — 퇴근 직전 하루를 마무리합니다.
작성 항목 상세 보기
Feedback 시작 — 오늘 시작은 어땠는지, 아침 루틴과 에너지 수준을 돌아봐요.
Feedback 중간 — 오전/오후를 나눠서, 어디서 집중했고 어디서 흐트러졌는지 기록합니다.
Feedback 마무리 — 하루를 마무리하며 나에게 한마디. 칭찬, 반성, 다짐 모두 좋아요.
내일 기상 직후 할 일 — 내일 아침 모닝 탭에 자동으로 표시됩니다. 잠들기 전 뇌에 미리 과제를 심어두는 효과.
한 줄 요약 — 오늘 하루를 한 줄로 압축. 나중에 히스토리에서 한눈에 볼 수 있어요.

입력한 내용은 자동으로 저장됩니다 (브라우저 + entries/ 폴더). 히스토리 탭에서 과거 기록을 확인하고, Claude Code의 주간/월간 리뷰 스킬로 AI 분석을 받을 수 있어요.

Claude Code 스킬
/diary-save
다이어리 마크다운을 entries/ 폴더에 파일로 저장하고 git commit합니다.
트리거: "다이어리 저장", "diary save", 마크다운 붙여넣기
/weekly-simol
이번 주 엔트리를 분석해 Big 3 달성률, 패턴, 정체성 추적, 다음 주 액션을 제안합니다.
트리거: "SIMOL 주간 리뷰", "weekly simol"
/monthly-simol
해당 월 엔트리와 주간 리뷰를 종합해 성장 곡선, 정체성 변화, 다음 달 방향을 분석합니다.
트리거: "SIMOL 월간 리뷰", "monthly simol"
/quarterly-simol
3개월 엔트리와 월간 리뷰를 종합해 정체성 진화, 미래 시각화 달성도, 큰 그림을 점검합니다.
트리거: "SIMOL 분기 리뷰", "quarterly simol"
성과평가 연결

매일 Big 3에 태그를 붙이면, 성과평가 시즌에 AI Nativeness 셀프리뷰를 자동으로 생성할 수 있어요.

Big 3 태그 활용법
업무 일반 업무 항목 AI활용 Claude Code 등 AI 도구를 활용한 업무 팀기여 팀/조직에 기여한 업무
마크다운에 #AI활용 형태로 저장되어, /ai-review 실행 시 자동으로 수집됩니다.
데이터 흐름
매일 Big 3 작성 (태그 선택) ↓ 통계 탭에서 AI활용/팀기여 일수 확인 ↓ /ai-review 실행 → Git + SIMOL 데이터 분석 ↓ 셀프리뷰 초안 자동 생성 (Before/After 포함)
/ai-review
Git 커밋 + SIMOL 엔트리를 분석해 AI Nativeness 셀프리뷰 초안과 점수 시뮬레이션을 자동 생성합니다.
트리거: "셀프리뷰", "AI 리뷰", "성과평가 준비"
